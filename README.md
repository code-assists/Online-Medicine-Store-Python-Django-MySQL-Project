# 💊 Online Medicine Store | Python, Django & MySQL Project  

**Online Medicine Store** is a **full-stack Python Django project with MySQL database**.  
It manages **Medicines, Medicine Types, Companies, Customers, Orders, Items, and Order Status Updates**.  

The project has **two types of users**:  
- **Admin** → Manage medicines, customers, companies, orders, and reports.  
- **Customer** → Register, browse medicines, place orders, and track order history.  

This is a **major Django project** for **final year computer science students (B.Tech, MCA, BCA, M.Tech)**. It comes with **source code, database, and documentation**.  

---

## 📖 Abstract  

The **Online Medicine Store System** automates medicine sales and management.  
- **Admin** → Can add, edit, delete, and view all medicines, customers, orders, and companies.  
- **Customer** → Can view medicines, order them, and check order status.  

This project is an excellent choice for students looking for **Python Django MySQL projects with source code**.

## 📂 Modules & Features  

### 👤 Customer Module  
- Admin can add, edit, update, and delete customers.  
- Customers can view & update their details.  
- Client-side form validation with JavaScript.  

### 💊 Medicine Module  
- Admin can add, edit, delete, and view medicines.  
- Customers can browse available medicines.  

### 🧾 Medicine Type Module  
- Admin can manage medicine categories/types (CRUD).  
- Customers can view medicine types.  

### 🏭 Medicine Company Module  
- Admin can manage medicine companies (CRUD).  
- Customers can view medicine companies.  

### 🛒 Order Module  
- Admin manages all orders.  
- Customers can place & view their orders with receipts.  

### 📦 Item Module  
- Admin manages order items.  
- Customers can view purchased items.  

### 📌 Order Status Update Module  
- Admin updates order statuses (CRUD).  
- Customers can track their orders.
